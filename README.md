# Automated Balance Sheet Generator

A web-based financial analysis tool that generates structured balance sheets dynamically using real company ticker symbols.

Built using Python and Streamlit.

---

## Features

- Accepts any valid company ticker (e.g., AAPL, MSFT, TSLA)
- Automatically fetches financial data
- Generates a structured balance sheet
- Clean and interactive web UI
- Fast execution

---

## Tech Stack

- Python
- Streamlit
- Financial Data APIs
- Pandas

## How to Run

1. Clone the repository:
   git clone https://github.com/YOUR_USERNAME/automated-balance-sheet-generator.git

2. Install dependencies:
   pip install -r requirements.txt

3. Run the app:
   streamlit run app.py
